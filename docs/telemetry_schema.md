# Telemetry Schema

The aircraft sends telemetry packets with the following fields:

- timestamp_ms
- packet_id
- node_id
- accel_x_g
- accel_y_g
- accel_z_g
- gyro_x_dps
- gyro_y_dps
- gyro_z_dps
- temperature_c
- pressure_hpa
- altitude_est_m
- battery_voltage_v
- low_power_flag
- gps_lat_deg
- gps_lon_deg
- gps_alt_m
- gps_speed_mps
- gps_fix_status
- satellite_count
- link_status
- mode_state
